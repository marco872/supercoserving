Real Estate Development Stakeholders use the following BNB Smart Contracts based on Monitary Policy

Important note:

Smart Contracts are authomatically executed when stakeholders' obbligations are signed and registered.

SC.1 Government: exchange Fiat into FBM tokens for selected project
SC.2 Investor: create Liquidity Pool between FBM and BNB tokens for selected project.  Liquidity Pool to be used by Property Owner to build selected project
SC.3 Property Owner: exchange Fiat selected project revenue into BNB tokens
SC.4 Tenant: exchange Fiat rent in BNB tokens for selected project
SC.5 Tenant: transfer BNB selected project rent tokens into Property owner wallet
SC.6 Property Owner: return BNB selected project borrowed tokens plus interests into Liquidity Pool
SC.7 Property Owner: return BNB selected project profit tokens into Government Wallet in FBM tokens
SC.8 Government: return 20% of FMB selected project profit tokens into Co-Serving Wallet
SC.9 Government: retain 40% of FMB selected project profit tokens into Government Wallet
SC.10 Government: return 20% of FMB selected project profit tokens into Property Owner Wallet
SC.11 Government: return 20% of FMB selected project profit tokens into Tenant Wallet
